# Printer Troubleshooting

## Printer Offline

- Verify power
- Check USB or network connection
- Restart printer
- Set as default printer
- Clear print queue

---

## Paper Jam

- Turn off printer
- Remove jammed paper carefully
- Check rollers
- Restart printer

---

## Poor Print Quality

- Check ink or toner
- Run print head cleaning
- Align print heads
- Replace cartridge if necessary

---

## Driver Issues

- Remove old driver
- Install latest manufacturer driver
- Restart Print Spooler service if required

---

## Preventive Maintenance

- Keep printer clean
- Use recommended paper
- Replace consumables on schedule
