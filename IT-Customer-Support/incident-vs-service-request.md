# Incident vs Service Request

| Incident | Service Request |
|-----------|-----------------|
| Something is broken | User needs a service |
| Requires troubleshooting | Usually pre-approved |
| Unexpected issue | Planned request |
| Restore normal operation | Fulfill user request |

## Examples of Incidents

- Blue Screen
- Wi-Fi disconnected
- Outlook crashes
- Printer offline

## Examples of Service Requests

- Install Microsoft Office
- Create user account
- Reset password
- Request VPN access
- Install printer
